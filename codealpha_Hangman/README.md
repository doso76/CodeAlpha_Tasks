# Hangman Game (Python)

A console-based implementation of the classic Hangman game built using Python. This project demonstrates clean program structure, input validation, and modular design principles.

## Features
- Random word selection
- Input validation (single-letter enforcement)
- Repeated guess detection
- Dynamic word display updates
- Game state tracking with limited lives
- Clean function-based architecture

## Concepts Demonstrated
- Functions and modular programming
- Loops and conditional logic
- String manipulation
- Lists and indexing
- Basic game state management

## How to Run

1. Clone the repository:
   git clone https://github.com/doso76/codealpha_Hangman.git

2. Navigate into the project directory.

3. Run the program:
   python hangman.py
