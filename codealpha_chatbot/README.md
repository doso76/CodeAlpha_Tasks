# Simple Python Chatbot 🤖

A beginner-friendly **rule-based Python chatbot** that responds to greetings, common questions, and farewells.  
This project demonstrates clean Python code, input handling, and simple keyword-based AI logic.

---

## Features
- Responds to greetings like "hi", "hello", "hey", "yo"
- Handles common questions like "how are you", "what is your name", "who created you"
- Responds to farewells like "bye" and "goodbye"
- Recognizes polite phrases like "thank you" and "good morning/afternoon/evening"
- Easy to extend with new phrases or keywords
- Console-based interactive chat

---

## Concepts Demonstrated
- Functions and modular design
- Conditional statements and logic
- String manipulation and input cleaning
- Basic rule-based NLP concepts
- Scalable, readable code structure
